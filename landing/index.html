How It Works

Implementing real-time collaboration has never been easier.

1

Install the SDK

Add our SDK to your project with npm or yarn and configure with your API key.

npm install realtimecursor-sdk
# or
yarn add realtimecursor-sdk
2

Initialize the Client

Set up the client with your API credentials and connect to your projects.

const { client } = useRealtimeCursor({
  apiUrl: 'https://api.realtimecursor.com',
  socketUrl: 'https://api.realtimecursor.com'
});
3

Add Components

Use our React components to add real-time collaboration to your app.

<CollaborativeEditor
  content={content}
  onChange={handleChange}
  editorRef={editorRef}
/>
